# Superstore Sales Dashboard (Excel Project)

This project analyzes the Superstore dataset using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Dashboard Design

## Analysis Performed
- Sales by Region
- Profit by Category
- Monthly Sales Trend
- Sales by Segment

## Sheets in the File
1. Sample Store – Contains the dataset
2. Dashboard – Contains charts and visualizations
